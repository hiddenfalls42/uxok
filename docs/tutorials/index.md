# Tutorials

Tutorials lead you through complete, hands-on lessons that end in a working, runnable result. The tutorial in this section teaches the uxok patterns that matter most by having you build something real — not by explaining concepts in the abstract, but by walking you through code that you can run and verify yourself. Start here if you have not used uxok before.

## In this section

- [Getting started](getting-started.md) — Build a minimal host with two plugins, wire a capability between them, and run the full lifecycle from `Core.start()` to `Core.stop()`.
